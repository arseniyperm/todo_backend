# Бэкэнд приложения "Список дел" на Python с помощью FastAPI.

## Функционал

**Аутентификация**
- Регистрация: POST /auth/sign-up - создание нового пользователя
- Вход: POST /auth/sign-in - авторизация
- Профиль: GET /auth/user - получение информации о текущем пользователе

**Управление задачами**
- GET /todos/ - список всех задач
- POST /todos/ - создание новой задачи
- GET /todos/(todo_id) - получение конкретной задачи
- PUT /todos/(todo_id) - обновление задачи
- DELETE /todos/(todo_id) - удаление задачи

## Запуск приложения

### Вариант 1: Запуск через Docker

Убедитесь, что у вас установлен Docker

Запустите Docker, откройте в нем терминал и выполните команду
```
docker pull arseniyperm/tset_todo-web
```

Далее запустите контейнер из импортированного Image с параметрами:
- Host port: 8000
- Environment variables (_variable:_ jwt_secret, _value:_ your_secret_key)

Документация приложения будет доступно по адресу: [http://localhost:8000/docs](http://localhost:8000/docs)

### Вариант 2: Локальный запуск

После загрузки данного репозитория откройте проект в своем IDE.

Создайте .env файл, в котором надо указать
```
JWT_SECRET=your_secret_key
```
Установите необходимые пакеты:.
```
pip install -r requirements.txt
```


**Использование Redis**
В репозиторий включена папка Redis-x64

Для логирования и кэширования в Redis, запустите файл **redis-server.exe**


Для запуска можно использовать конфигурацию запуска:
-запускать модуль **todo**, указав рабочую директорию и путь к **.env** файлу
-запускать файл **src/todo/__main__.py**, указав путь к **.env** файлу

Документация приложения будет доступно по адресу: [http://localhost:8000/docs](http://localhost:8000/docs)

Логи операций можно посмотреть в файле **logs/todo_service.log** приложения
